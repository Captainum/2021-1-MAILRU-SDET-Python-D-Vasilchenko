# Подсчет топ 5 пользователей по количеству запросов, которые завершились серверной (5ХХ) ошибкой

## Bash версия
#### Использование:
```Bash
    top_requests_serv.sh <inputfile> <outputfile> [-h]
```
#### Описание работы:
* Оставляем в строках только *ip* и статус запроса
* Оставляем только строки с серверной ошибкой
* Оставляем в строках только *ip*
* Сортируем строки в лексикографическом порядке
* Подсчитываем количество пользователей с помощью
```Bash
    uniq -c
```
* Сортируем их по количеству запросов
* Выводим первые 5 значений в *outputfile*

## Python версия
#### Использование:
```Bash
    python top_requests_serv.py [-h] -i INPUTFILE -o OUTPUTFILE [--json]
```

#### Описание работы:
* Считываем построчно файл *inputfile*
* Подсчитываем количество пользователей
* Сортируем их по количеству
* Выводим первые 5 значений в *outputfile*
